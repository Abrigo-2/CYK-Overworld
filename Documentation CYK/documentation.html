THIS IS NOT CREATE YOUR FRISK'S NOR SIMPLY CYK'S DOCUMENTATION!!!

    If you're reading this, then it means you're using the Create your Kris's + Overworld's fork. For ease of use, some modifications have been made to pre-existing variables and functions, so check the further below in order to know which. These descriptions have also been added throughout the documentation, so worry not!

    Welcome to Create Your Kris's documentation!



    You probably want to get right in, and it may be tempting to simply check out the existing encounters and tinker with them. You're free and encouraged to do so.

    However, there is certain information about how the engine operates that you'll be better off knowing from the get-go. This is what the Setup and Usage guide is for. You'd do well to read it, in order to prevent headaches in the long run!

    Also! Sections starting with a * can be collapsed by clicking on them. Try it out on these ones!

    Explains the folder and file structure of CYK and mods created with it.

    The Player Object
    The Player object now only controls the red soul you use in the menus.
    The new Entity Object controls Player characters.

    The Entity Object
    This section details everything you can do with the new Entity objects in CYK.

    The Script Object
    Script objects are no longer used for Monster scripts.

    Wave Scripts
    The Arena now has 3 new properties.

    The Inventory Object
    Created items can now target players or monsters as you please!

    Projectile management
    Bullets have some extra variable properties, and the function OnHit has been significantly altered.
