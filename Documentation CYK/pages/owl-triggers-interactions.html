Interactable (Triggers)


    Properties



    dialogueID This Trigger's Dialogue Tag (as referred to in the rest of the documentation.)

    When the player interacts with it (by pressing the Confirm key), this Trigger "sends" its current dialogueID to the Overworld's Overworld.HandleInteractable() Event function, which then performs an action regarding the tag.

    The default action is to create a Textbox!
    More on this over at the Textboxes and Dialogues tab.


    dialoguesTotal = 1 If set above 1, each time the player interacts with this object, an internal dialogue counter will go up by one (but will never exceed this value.)

    Then, each following time this object is interacted with, it will "send" the tag immediately followed by its internal dialogue counter.

    Assuming that dialoguesTotal = 3 and dialogueID = "tag", this would happen:

    1. First interaction: Sends "tag".
    2. Second interaction: Sends "tag-2".
    3. Third interaction: Sends "tag-3".
    4. Fourth (and following interactions): Sends "tag-3".

    This counter will be reset upon leaving the room.
