- Savepoints (Triggers)



    Savepoints will only store the following data:

    • Overworld.story
    • Overworld.storyFlags

    • All Avatars' max HP (and, by implication, their current hp.)

    • The currently playing BGM (If none is playing, it also notes that down.)
    • The current room's Name.
    • The current Savepoint's location value.


    Loading of this data is then taken care of by your Encounter script's EncounterStarting() event function (assuming you haven't tampered with the code...!)

    Properties






    location The "name" of this Savepoint. This is the name of the location displayed in the Save Screen.


    hasText = true If true, creates a Textbox directly, (without calling Overworld.HandleInteractable) using the current room Dialogue 's tag "save".
