Controls



    These are mostly the same as in Undertale/Deltarune, save some added debugging controls.
    For the sake of consistency, it's encouraged to respect said control scheme in your mods.


    Arrow/WASD keys Movement, for the Arena, UI, and the Overworld.


    Z (or Enter) These act as the "Confirm" key.


    X (or left/right Shift) These, in turn, act as the "Cancel" key.


    C This allows for skipping over any Battle flavor text, speech bubbles, or Overworld textboxes.
    Note: No game menu has been made yet, so don't expect anything else from this key.


    F4 / Alt+Enter Toggles Fullscreen on and off.


    Esc Closes the current mod and boots back the Mod Selection screen.
    : if the encounter has "unescape" set to true, a "Quitting" prompt resembling that of Undertale will be shown before closure. The library code for this can be fiddled with.


    F9 Toggles the debug console. You can write text to this with DEBUG("your text here") in your scripts.


    H If the debug console is active, this'll show the hitboxes for the bullets and the player in the Arena.
