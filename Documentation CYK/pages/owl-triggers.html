What are Triggers?



    "Triggers" are objects that perform a particular action once the player's Avatar interacts with them.

    This interaction can ocurr by:

    • The Avatar walking inside their bounding box:
    • The player pressing the Confirm button whilst the Avatar is right next to them:

    Triggers must be placed into the respective Triggers layer via the Ogmo Editor

    The following function may be called within the Overworld instance.
    That is, you may call it as Overworld.FunctionName(),
    or as self.FunctionName() if inside the Overworld/RoomHandler script.



    object Overworld.FindObjectInRoom( ogmoLayer, ogmoObjectID) Returns the object from the given ogmoLayer with the given ogmoObjectID, within the current room.

    • Keep in mind that, although Triggers contain Sprites, Triggers aren't sprites themselves.
    • Assets, on the other hand, are always Sprites.

    You can find which id belongs where in Ogmo.
