The + Overworld Library - Overview



    Let's not waste any time:

    The +Overworld Library includes the following, and only the following features:

    • Multiple encounter loading.
    • Textboxes and dialogues outside of Encounters.
    • Room switching and generation. (via the OgmoEditor tool)
    • Tileset generation. (also via the OgmoEditor)
    • Saving and Loading player's session data.


    If a desired behavior wasn't mentioned above, the library simply does not have it.

    You'll have to implement it yourself.


    This library is not meant to compete, overshadow, nor be an equal to the Kristal engine.

    Check out each respective section for further details.

    Some may want to use only certain features of the library (such as the combat polish, loading multiple encounters or the cutscenes interface). For those, I've written the Optimizing for 1 Encounter page, as well as the Encounter Only.lua demo.

    Also, please make sure you've read the repository's README.md, as it credits features that Weren't Just Me(tm), as well as a certain compatibility issue.
